FREEZE - Stockage Cloud Intelligent

Conception d un service serverless de retention des fichiers avec URI stable et optimisation dynamique des couts de stockage AWS.

RoleArchitecte Cloud / Developpeur
ContexteAirbus
DomaineStorage Engineering
StackLambda, API Gateway, S3, DynamoDB, EventBridge

Contexte

Le besoin etait d offrir un acces stable aux fichiers tout en optimisant les couts. Les classes S3 devaient etre exploitees automatiquement selon l usage reel des donnees.

Solution technique

  • API Gateway + Lambda pour exposer les operations de creation, suppression et resolution.
  • DynamoDB pour porter les metadonnees de cycle de vie et localisation logique.
  • Mecanisme quotidien via EventBridge pour migration chaud / tiede / froid.
  • Infrastructure complete deployee avec AWS CDK pour repetition sans ecart.

Resultats

Couts maitrises

Reduction des depenses stockage grace aux classes adaptees a l usage.

URI stable

Les clients accedent aux fichiers via une adresse logique unique.

Scalabilite serverless

Absorption de charge sans gestion manuelle d infrastructure.
