Application de Gestion des Ressources

Developpement d une application de gestion RH avec backend Java robuste et interface Angular pour suivi et allocation des ressources.

RoleDeveloppeur Full-Stack
ContexteProjet professionnel
DomaineGestion operationnelle
StackSpring Boot, PostgreSQL, Angular, REST

Contexte

Le suivi des ressources etait disperse, ce qui compliquait la planification et le pilotage des charges.

Solution technique

  • Backend Spring Boot structure en couches avec API REST.
  • Persistance PostgreSQL et modeles relationnels metier.
  • Frontend Angular pour gestion et visualisation des allocations.

Resultats

Centralisation

Vision unifiee des ressources et affectations.

Productivite

Moins d operations manuelles et suivi plus rapide.

Base evolutive

Architecture solide pour ajout de fonctionnalites futures.
