Pipeline Meteo de Bout en Bout

Conception d un pipeline ETL batch complet pour ingerer des donnees meteo depuis une API publique, les transformer, les modeliser en entrepot local et les visualiser dans un dashboard interactif.

RoleData Engineer Junior
TypeProjet Personnel
NiveauOriente Production
StackPython, Pandas, DuckDB, SQLite, Metabase

Description

Ce projet met en place un flux data engineering de bout en bout: ingestion API, stockage brut, transformation analytique, chargement en entrepot et exposition BI. L objectif est de reproduire un environnement local proche des pratiques de production.

Responsabilites Principales

  • Developpement d un script Python d ingestion pour collecter des donnees meteo depuis une API publique.
  • Mise en place du stockage brut en JSON (couche data lake).
  • Developpement de la logique de transformation avec Pandas.
  • Chargement des donnees curiees dans un data warehouse SQLite.
  • Connexion du warehouse a Metabase pour la business intelligence.
  • Conception de dashboards interactifs sur les tendances de temperature et d humidite par ville.

Points Forts de l Architecture

  • Pipeline d ingestion en batch.
  • Architecture data en couches: raw -> transformed -> warehouse -> BI.
  • Environnement local reproductible avec virtualenv.
  • Design de pipeline modulaire et extensible.

Stack Technique

  • Python, Pandas, DuckDB, SQLite.
  • Metabase, API REST, Modelisation de donnees.

Valeur Metier

Ce projet demontre la capacite a construire des pipelines data proches de la production, transformer des donnees semi-structurees, modeliser des jeux analytiques et livrer des insights actionnables via des dashboards BI.
