Le Problème

L'analyse vidéo dans le football nécessite traditionnellement de nombreux opérateurs humains pour taguer manuellement les actions. Les systèmes automatiques existants nécessitent souvent des caméras fixes coûteuses ou des capteurs GPS sur les joueurs.

Ma Solution

J'ai développé un pipeline complet capable de traiter n'importe quelle vidéo de match (broadcast) :

  • YOLOv8 pour la détection d'objets (Joueurs, Ballon, Arbitre).
  • ByteTrack pour suivre les joueurs même lorsqu'ils se croisent (Tracking ID).
  • K-Means Clustering pour analyser les couleurs des maillots et assigner automatiquement les équipes.
  • Transformation Homographique pour projeter les positions sur un terrain 2D virtuel.
