Étude de Cas : Maintenance Prédictive pour Compresseurs Industriels avec le Machine Learning

Développement d'une solution de bout en bout pour prédire la performance des compresseurs et optimiser la production d'acide sulfurique chez OCP Group.

  • RôleData Analyst & Développeur Machine Learning
  • EntrepriseOCP Group
  • Durée3 mois
  • TechnologiesPython, Scikit-learn, Pandas, Flask [cite: 41, 44, 53]
❮ ❯

Le Contexte : Un Équipement Stratégique

Au sein de la ligne de production d'acide sulfurique du groupe OCP, le compresseur radial Siemens de type SFO est un équipement critique. Sa performance conditionne directement l'efficacité et la fiabilité de tout le processus industriel. Une défaillance ou une simple baisse de rendement peut entraîner des arrêts de production coûteux et des pertes d'efficacité énergétique.

La Problématique : De la Maintenance Réactive à Prédictive

Le principal défi était la surveillance et la gestion des variations des paramètres critiques du compresseur (débit, pression, température). Des fluctuations non contrôlées peuvent indiquer une dégradation progressive, menant à des pannes imprévues. L'objectif de ce projet était de passer d'une maintenance réactive (réparer après la panne) à une approche prédictive, en anticipant les besoins de maintenance avant qu'une dégradation significative ne se produise.

Ma Solution : Une Application de Machine Learning Complète

Pour résoudre ce problème, j'ai développé une solution complète de machine learning, de l'analyse des données au déploiement d'un outil accessible aux équipes opérationnelles. La démarche s'est articulée autour de trois axes :

  • Analyse et Modélisation : J'ai analysé un historique de 24 mois de données de production. J'ai ensuite entraîné un modèle de régression de type Random Forest sur ces données pour sa robustesse et sa grande précision.
  • Développement de l'API : Pour rendre le modèle utilisable, j'ai développé une API RESTful avec Flask. Cette API sert de pont entre les données d'entrée et le modèle de machine learning.
  • Interface Web Utilisateur : J'ai créé une interface web simple où les techniciens peuvent télécharger un fichier Excel contenant les derniers relevés du compresseur et obtenir en retour une prédiction instantanée de sa performance en pourcentage. Le code source est disponible sur mon dépôt GitHub.

Résultats & Impact

Modèle Haute Précision

Le modèle a atteint un coefficient de détermination (R²) de 0.9997, signifiant qu'il explique la quasi-totalité de la variance des données et fournit des prédictions très fiables.

Maintenance Prédictive

L'outil permet d'anticiper la dégradation des performances, donnant aux équipes les moyens d'intervenir avant une panne et de planifier la maintenance.

Outil Opérationnel

La solution n'est pas restée un simple notebook : une application web fonctionnelle a été livrée, la rendant directement utilisable par les équipes sur site.

Fiabilité Améliorée

En fournissant des prédictions en temps réel, l'outil contribue à améliorer la fiabilité du processus de production et à réduire les coûts opérationnels liés aux arrêts imprévus.
