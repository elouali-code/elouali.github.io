Industrialisation DevOps ARDmate

Transformation d'un prototype en chaîne d'outils industrielle chez Airbus. Mise en place de pipelines CI/CD, tests automatisés et refonte architecturale.

RôleIngénieur DevOps
EntrepriseAirbus
Tech StackPython, Jenkins, Git, Pytest
❮❯

Le Contexte : Prototype Instable

L'outil existant était un script monolithique de 2000 lignes, sans tests ni documentation. Les mises en production étaient risquées et manuelles, provoquant des erreurs dans la génération des documents d'architecture critiques.

Ma Solution : Refonte & Automatisation

J'ai mené une refonte complète en appliquant les standards industriels du développement logiciel :

  • Refactoring : Découpage du code en modules réutilisables avec le pattern Composite.
  • Testing : Écriture de 130+ tests unitaires (Pytest) pour garantir la non-régression.
  • CI/CD : Création de pipelines Jenkins automatisés pour le build, le test et le packaging.
  • Collaboration : Mise en place du Git Flow et des Conventional Commits.

Résultats & Impact

Fiabilité Totale

Couverture de tests > 90%, éliminant les bugs critiques en production.

Maintenance Facile

Code modulaire et documenté, réduisant le temps de correction de 50%.

Qualité Garantie

Validation automatique de chaque commit avant fusion.
